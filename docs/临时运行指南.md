# 临时运行指南

## 修改配置
1. 修改*\src\main\resources\application.yml*中
    ```yaml
    spring:
        datasource:
            url: jdbc:mysql://localhost:3306/sms?useSSL=false
            username: root
            password: 135246qq
    ```
   中的`username`和`password`为你自己的数据库用户名和密码。

## 构建并运行项目
1. 双击*\script\build.bat*，等待构建完成，随后将会自动运行，如果构建失败，请检查错误信息。
